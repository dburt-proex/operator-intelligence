# Assets

This directory stores reusable visual and brand assets for Operator Intelligence reports.

## Planned assets

- Report cover layouts
- Scorecard components
- Operator Score gauge
- Maturity tier visuals
- Roadmap diagrams
- Customer journey diagrams
- Revenue leakage diagrams
- Icons
- Brand palette
- Typography rules

## Design direction

Use a premium consulting-report aesthetic:

- Executive-friendly
- Minimal
- Structured
- High-contrast
- Data-forward
- No gimmicks

Preferred visual tone:

- Black
- Silver
- Muted gold
- White space
- Blueprint/system-map accents

## Rule

Visual design should clarify the assessment. It should never distract from evidence, scoring, or recommendations.
