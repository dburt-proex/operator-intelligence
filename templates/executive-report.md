## 13. Release checklist

- [ ] Report ID, version, methodology, evidence snapshot, and supersession state are fixed.
- [ ] QC and publication decisions resolve.
- [ ] Score display matches coverage, confidence, bounds, and publication state.
- [ ] Findings are evidence-backed and ledgered.
- [ ] Recommendation priority inputs reproduce.
- [ ] Package eligibility precedes assignment.
- [ ] Phase 0 and phases 1–5 are correctly used.
- [ ] Unknowns, blockers, assumptions, and limitations remain visible.
- [ ] Publication, roadmap approval, proposal acceptance, and implementation authorization remain separate.
- [ ] Completion and realized-value evidence remain separate.
- [ ] Executive-safe language review passes.
