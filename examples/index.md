## 3. Demonstrated controls

- synthetic evidence is clearly labeled
- active category weights total 100%
- unknown is not zero
- confidence is separate from maturity and priority
- social unknown creates no social finding
- package eligibility precedes assignment
- validation recommendations remain unrouted in Phase 0
- three eligible recommendations use one primary package each
- AI implementation is blocked
- provisional publication is separate from QC
- proposal acceptance and implementation authorization remain unset
- completion does not imply realized value

## 4. Usage

Use the folder to:

- train evaluators on the complete record chain
- regression-test schema and governance changes
- demonstrate commercial deliverables without exposing real client data
- validate future template, playbook, scoring, or package changes

Do not use any sample statement as evidence about a real company or market.
